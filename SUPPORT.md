# Support

SimuLoom is a community-maintained open-source project.

- Use GitHub Discussions for usage questions, design ideas, and examples.
- Use the bug report template for reproducible defects.
- Use the feature request template for scoped enhancements.
- Use GitHub private vulnerability reporting for security issues.

Include the SimuLoom version, runtime adapter, deployment method, sanitized logs, and a minimal
synthetic contract or scenario when requesting help. Never post credentials, production payloads,
customer contracts, protected health information, or other sensitive data.

There is currently no guaranteed response time or commercial support agreement.
